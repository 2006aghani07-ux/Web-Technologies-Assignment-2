# Recipe App 

# PREPARED BY: *ABDUL GHANI* *2502129*

The Recipe App is a web-based application built with HTML, CSS, and JavaScript. 
Its purpose is to provide an interactive platform for searching, filtering, and managing recipes. 
The app is designed to be user-friendly and works on both desktop and mobile devices.

Key Features:
Search recipes by name or ingredients.
Filter recipes by categories: Breakfast, Lunch, Dessert.
View recipe instructions in a collapsible section.
Add recipes to a favorites list that persists using localStorage.
Remove recipes from favorites at any time.
Technical Highlights:
Dynamic DOM manipulation with JavaScript.
Event handling for buttons, checkboxes, and input fields.
Responsive design using CSS.
Clean layout with structured HTML.

How to Run:
Open the project folder in Visual Studio Code.
Install the Live Server extension if not already installed.
Open index.html in VS Code.
Right-click → Open with Live Server to view the app in the browser.

Project Structure:
recipe-app/
├── index.html      # Main HTML page
├── style.css       # Styling
├── script.js       # JavaScript functionality
└── README.md       # Project documentation

Learning Outcomes:
Understanding client-side storage with localStorage.
Implementing dynamic DOM manipulation.
Designing responsive layouts.
Integrating search and category filtering.
The Recipe App demonstrates how to build a practical, interactive, and user-friendly web application from scratch.
